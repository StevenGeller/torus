Torus is semasiographic — it encodes meaning directly, not speech sounds. The same semantic relationships in any word order produce the same symbol.

"Dog bites man" and "man bites dog" produce different symbols because the agent and patient roles differ. But rearranging words without changing who-does-what-to-whom preserves the symbol.

Actions carry no tense. Only aspect: timeless, unbounded, or bounded. Modifiers attach to specific words. Start reading anywhere.
